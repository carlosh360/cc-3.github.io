Ejercicio 1

Una de las cosas que aconsejamos en este curso es haber llevado y aprobado CC2 “Estructura de Datos”, ya que por lo general necesitamos que tengan bastante experiencia en algún lenguaje de programación basado en C como “Java”. Muchas veces pasa que los alumnos que no tienen estas capacidades se sienten incapaces de llevar el ritmo del curso y terminan reprobándolo. Los siguientes dos problemas están diseñados para verificar que están listos para el material de CC3.

Ejercicio 1.1: Fibonacci

Escriban una función que, dado un numero n, retorne el enésimo numero de la serie de Fibonacci. La función está dada por el código de abajo: 

    // Retorna el n-esimo numero de Fibonacci. Asuman que n >= 0.
    public static int fibonacci(int n) {

        // Su codigo aqui

    }

La serie de Fibonacci está definida por lo siguiente: 

    Fib0 = 0
    Fib1 = 1
    Fibn = Fibn-1 + Fibn-2        Donde Fibn es el n-esimo numero de Fibonacci.

La lista de los primeros 10 números de Fibonacci son los siguientes: 

    Fib0 = 0
    Fib1 = 1
    Fib2 = 1
    Fib3 = 2
    Fib4 = 3
    Fib5 = 5
    Fib6 = 8
    Fib7 = 13
    Fib8 = 21
    Fib9 = 34

Ejercicio 1.2: Zipper

Escriban una función que dado dos arreglos, cada uno ordenado de menor a mayor, retorne un nuevo arreglo que contenga todos los elementos de ambos arreglos ordenados de menor a mayor. Por ejemplo, si los arreglos de entrada son los siguientes: 

    Arreglo de entrada 1 = { 1, 4, 7, 8, 8 , 9}
    Arreglo de entrada 2 = { 2, 4, 6, 8}

La salida debería de ser un arreglo que contenga los elementos en el siguiente orden: 

    Arreglo de salida = { 1, 2, 4, 4, 6, 7, 8, 8, 8, 9 }
